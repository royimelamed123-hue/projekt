# מעקב הרגלים — תוסף לאוצריא

תוסף לאפליקציית **אוצריא** למעקב יומי/שבועי/חודשי אחר הרגלים בלוח שנה עברי.

---

## תיאור

התוסף מאפשר:
- הגדרת הרגלים מסוגים שונים: יומי (`x_times`), שבועי (`weekly`), חודשי (`monthly`)
- מעקב לפי תאריך עברי
- סטטיסטיקות ואחוזי הצלחה לכל חודש
- הערות יומיות לכל הרגל
- גיבוי וייבוא נתונים
- מצב לילה (dark mode)
- נעילת גישה בסיסמה

---

## דרישות מערכת

- אפליקציית **אוצריא** (גרסה התומכת בתוספים)
- דפדפן מודרני (Chrome / Edge / Firefox)
- אין תלות ב-Node.js או חבילות npm

---

## התקנה

1. העתק את תיקיית הפרויקט לתוך תיקיית התוספים של אוצריא
2. וודא ש-`manifest.json` מוגדר כראוי עם שם התוסף
3. הפעל מחדש את אוצריא — התוסף יופיע בתפריט התוספים

## הוראות פיתוח

### עריכת קוד
כל הקבצים הם JavaScript רגיל (ES6+) ללא build step. ניתן לערוך ישירות.

### מטמון סטטיסטיקות
הפרויקט משתמש ב-3 מפות מטמון (מוגדרות ב-`storage.js`):
- `statsCache` — תוצאות `calculateStatsForMonth` לפי הרגל+חודש
- `totalAvgCache` — ממוצע כולל לפי הרגל
- `streakCache` — רצף נוכחי לפי הרגל

**חשוב:** בכל שינוי נתונים קוראים ל-`invalidateStatsCache(habitId)` כדי לנקות את המטמון.

### הוספת סוג הרגל חדש
1. הוסף את הסוג ל-`configSelectedType` options ב-`modals-config.js`
2. טפל בו ב-`setStatus` (`habits-logic.js`)
3. הוסף רינדור ב-`renderHabits` (`render.js`)
4. הוסף חישוב סטטיסטיקות ב-`calculateStatsForMonth` (`interactions.js`)
